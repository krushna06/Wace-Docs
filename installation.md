# Installation



### Bot installation:

1. Rename `example.app.yml` to `app.yml` and fill it with your credentials.
2. Start the bot using `npm run start.`

### Lavalink installation:

1. Install java 17.
2. Git clone this repo: [https://github.com/krushna06/Lavalink-Server](https://github.com/krushna06/Lavalink-Server)
3. Fill in the api/credentials inside `application.yml`.
4. Start the lavalink server using `java -jar Lavalink.jar`.
